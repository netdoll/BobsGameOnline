# IDEAS: bobsgameonlinejava

## 1. Feature Improvements
- **Random Sprite Output:** Procedurally generate block sprites for unique session aesthetics.
- **Move Map Up/Down:** Allow vertical shifting of the entire map in the editor or during special modes.
- **TWL Integration:** Explore replacing Scene2D with TWL for more complex desktop UI requirements.
- **Enhanced Undo:** Prevent undo-wrapping and intelligently skip redundant state snapshots (e.g., purely visual changes).

## 2. Refactoring & Porting
- **Kotlin Migration:** Consider a partial or full migration to Kotlin for better null safety and conciseness.
- **Capacitor Mobile:** Prototype a mobile wrapper for the Java logic using Capacitor or a similar bridge.

## 3. Project Pivot
- **Education Tool:** Pivot the engine to be a "Build your own Tetris" educational platform using the established parity logic as a reference.
